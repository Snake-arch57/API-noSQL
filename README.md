# API-noSQL
Neste projeto fiz um mini SIEM, onde você altera a prioridade, deleta incidentes e adiciona incidentes de uma forma simples e básica.

# Explicação
1. Para testar a API utilizei o Postman, para fazer as requisições e os seus metódos. 
2. Foram utilizados tambêm o MongoDB ATLAS que é uma solução em nuvem de banco dados. A API conecta diretamente nesse banco de dados e o altera, atualiza, deleta e lista todos os dados que estão no banco de dados noSQL
3. Na hora da criação do incidente, verifique se a tabela eventos está criado no seu MongoDB e se você está logado no mesmo.
